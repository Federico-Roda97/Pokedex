# Pokédex Project

## Overview
This project is a simple Pokédex web application built using HTML, CSS, and JavaScript. It retrieves Pokémon data from a public API and displays it in a user-friendly interface.

## Features
- Search Pokémon by name
- Display Pokémon information (name, image, stats)
- Responsive layout for different screen sizes
- Dynamic data fetching using a public API

## Technologies Used
- HTML
- CSS
- JavaScript
- Public Pokémon API

## API Used
This project uses a public Pokémon API to fetch data dynamically.

## Purpose
This project was created for learning purposes to improve skills in JavaScript, DOM manipulation, and API integration.
